Index Service

This service provides a facade to a Solr indexing server. A Solar instance is assumed to be found at:
  • Host: solr
  • Port: 32772
  • Core: docs

Endpoints

/

This information page.

/upload

Upload JSON or form data to the index. If the endpoint is accessed using GET, a form will be displayed which will post the following parameters:
  • id : A unique identifier for the document
  • title : A document title
  • content : The plain text version of the document
The document content will be fully indexed when uploaded. Application access to the service should typically not use the form itself, but POST the JSON representation of the above as the body, for example:

{"id" : "1234-1234-1234-1234", "title" : "A sample title", "content" : "Full text content goes here..."}

/find/:query

Returns a JSON list of documents matching the query. The underlying Solr implementation uses the edismax search algorithm which performs a "google-like" search. The :query parameter should be sent using the GET method
